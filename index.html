Project Overview

This site documents our COMPS project on compressing multispectral satellite images using JPEG and JPEG-2000.

What this project is about

This project compares how two image compression standards, JPEG and JPEG 2000, perform on multispectral satellite imagery. We implemented our own JPEG pipeline (color conversion, DCT, quantization, entropy coding) and a JPEG 2000 pipeline (wavelet transform, quantization, block coding), then evaluated their quality, bitrate, and runtime.

Dataset

We use a small set of multispectral satellite GeoTIFFs from the USGS Earth as Art collection. Each image is converted to an RGB view and then compressed and reconstructed with both JPEG and JPEG 2000 so we can compare quality and storage cost on the same settings.
